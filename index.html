What is the IPython Notebook?

Web-based interactive Python programming environment.

  • One document for code and output
  • Run locally and remote
  • Document process
  • Share results

Does not separate code and output.

  • Data structure ouput
  • Inline plots
  • Literate programming
    • Conversation, telling a data story
  • Great for iterative programming.
    • Data analysis
    • Quick scripts
    • Prototyping

Local and Remote

  • Run locally
  • Connect to the cloud (e.g AWS)
  • Connect to supercomputer (e.g. JANUS)
  • Add compute power:
    • mpi4py
    • IPython Parallel
    • spark MapReduce
    • Numbapro GPU
    • ...

import pandas as pd

def plot_res(res):
    num_engines = len(res.groupby('id').size())
    fig, ax = subplots(figsize=(10, 8))
        
    tmin = res['start'].min()    
    for i in res.index:
        x_start = res.ix[i]['start'] - tmin
        x_end = res.ix[i]['end_time'] - tmin - x_start
        x_id = float(res.ix[i]['id'])/4.0
        #print x_start, x_end
        
        ax.add_patch(Rectangle((x_start, x_id),x_end,0.24,alpha=0.5, color='green'))
    ax.set_xlim(0, 1.5)
    ax.set_ylabel("Worker")
    ax.set_xlabel("seconds")
